About

🌟 1.Introduction

The Book Recommendation Site is a web-based application that allows users to add, view, and manage book recommendations. Users can enter book details (title, author, genre), see a list of added books, and even remove books they no longer want on the list.

✨ Key Features

✅ Add books with title, author, and genre

✅ Display books in a structured list 📖

✅ Store books in Local Storage so they don’t disappear on refresh 🔄

✅ Fully responsive for mobile & desktop 📱💻
